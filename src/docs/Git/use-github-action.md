### [actions/checkout](https://github.com/actions/checkout)

- 워크플로에서 저장소에 액세스할 수 있도록 체크아웃 작업을 함.
- actions/checkout 을 사용하지 않으면 github action 워크플로에서 저장소의 데이터에 접근하지 못 함.
